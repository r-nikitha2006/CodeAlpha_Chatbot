# CodeAlpha FAQ Chatbot

This project was developed as part of the **CodeAlpha Artificial Intelligence Internship**.

## Project Overview
The FAQ Chatbot is a simple Artificial Intelligence application that answers user questions automatically. It uses Natural Language Processing (NLP) techniques to compare the user's question with a set of predefined FAQs and returns the most relevant answer.

This chatbot demonstrates basic AI concepts such as text processing, similarity matching, and conversational interaction.

## Features
- Answers frequently asked questions
- Uses Natural Language Processing (NLP)
- Finds the most similar question using cosine similarity
- Simple command-line chatbot interface

## Technologies Used
- Python
- NLTK
- Scikit-learn
- Natural Language Processing (NLP)

## How It Works
1. The chatbot contains a list of predefined questions and answers.
2. When the user asks a question, the system processes the text.
3. Cosine similarity is used to compare the user question with stored FAQs.
4. The chatbot returns the most relevant answer.
